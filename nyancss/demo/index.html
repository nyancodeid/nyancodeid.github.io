Getting Started

Nyan.css comes in two different forms. You can select which version you want depending on out preference and expertise. To start using Nyan.css, all you have to do is Download one of the options below.


Download

This is standart version that comes with both the minified and unminified CSS. This options requires little to not setup.

Nyan.css -version-

CDN

You can find all the version of the CDN at github.com/nyancodeid/nyancss

<!-- Compiled and Minified CSS -->
<link rel="stylesheet" href="-linkcss-" >

<!-- Compiled and Minified Javascript -->
<script src="-linkjs-" defer ></script>


Setup

Project Structure

After downloading, extract the files into the directory where your website is located. Your directory will look samething like this.

You'll notice that there are two sets of the files. The min means that the file is "compressed" to reduce load times. These minified files are useally used in production while it is better to use the unminified files during development. 

MyWebsite/
	|--css/
	|	|--nyan.css
	|	|--nyan.min.css
	|
	|--fonts/
	|	|--avenir/
	|
	|--js/
	|	|--nyan.js
	|	|--nyan.min.js
	|
	|--index.html


HTML Setup

Next you just have to make sure your link files properly in your webpage. Generally it is wise to import javascript files (ex. jQuery) at the end of the body to reduce page load time. And don't forget to add ny-app at body tag. Follow the example below on how to import Nyan.css into your webpage.

<!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="UTF-8">
	<meta http-equiv="X-UA-Compatible" content="IE=edge">
	<!-- Lets browser know website is optimized for mobile -->
	<meta name="viewport" content="width=device-width, initial-scale=1.0">

	<!-- Import Nyan.css -->
	<link rel="stylesheet" type="text/css" href="css/nyan.min.css" media="screen,projection">
</head>
<body ny-app>
	
	<!-- Import javascript file here -->
	<script src="vendor/js/jquery.min.js"></script>
	<script src="js/nyancss.min.js" defer ></script>
</body>
</html>

And here we go!

Open Menu

Open menu use for menu dropdown. make element div with class="ny-openmenu" as parent. and for child element, insert button (with attribut data-trigger="ny-openmenu" as openmenu trigger button) and .ny-openmenu--container for open menu container.

for Animation add attribut data-animation on .ny-openmenu--container

Attribut Value
data-animation left or right
data-position above or under

Openmenu using javascript code for run click event. Add this code when document ready. add outsideClick to close openmenu when clicked outside element. default outsideClick is false

$(document).ready(function() {
	...
	$.openMenu(); // default

	$.openMenu({
		outsideClick: true // close openmenu when click outside
	}); 
});

Accordion

Make div parent element with class="ny-accordion", and for Child Item Element add div with class="ny-accordion-item" and don't forget to add input and label element. For content add div element with class="ny.accordion--content" inside ny-accordion-item under input+label element.

There are 2 version of accordion, single open and multiple open. Use this parent class for they are :

Accordion Class
Single Open (* default) .ny-accordion
Multiple Open .ny-accordion--multiple

Add checked="checked" attribut at input checkbox / radio element for default opened accordion.

Tabs

If tab has attribut data-disabled / class .ny-tabs--disabled, then tab not be clicked.

Attribut of Tabs

Attribut Value Places Description
data-tabs-content byIndex parent .ny-tabs use for trigger tab head with tab container, byIndex is calling the content in which the content index and tab headers are both the same.
data-tabs-content byTarget parent .ny-tabs use for trigger tab head with tab container. byTrigger, call content which is where data-trigger in tab head and data-tabs in ny-tabs-container items.
data-default - child of .ny-tabs-header (li element) Default active tabs on webpage load
data-disabled - child of .ny-tabs-header (li element) Default disabled tabs on webpage load

Classes of Tabs

Class Places Description
ny-tabs--fullwidth .ny-tabs-header Full width tab head non scrolled
ny-primary .ny-tabs-header Display Primary Color Theme
ny-accent .ny-tabs-header Display Accent Color Theme
ny-tabs--disabled .ny-tabs-header list-item (li element) Class to disabled tabs

Media Queries

Nyan.CSS uses media queries to serve its scalable grid, But also has a list of queries for convenience of styling your site across devices. The sizes for the queries are:

  • Desktop HD : 1200px
  • Deskto : 1000px
  • Tablet : 750px
  • Phablet : 550px
  • Mobile : 400px
Source from getskeleton.com

@media (min-width: 400px) {}

/*Large than phablet*/
@media (min-width: 550px) {}

/*Large than tablet*/
@media (min-width: 750px) {}

/*Large than desktop*/
@media (min-width: 1000px) {}

/*Large than desktop hd*/
@media (min-width: 1200px) {}

Layout Align Classes (Child Alignment)

The .ny-layout-align--*-* class takes two words. The first word says how the children will be aligned in the layout's direction, and the second word says how the children will be aligned perpendicular to the layout's direction.

ny-flex--col ny-layout-align--center-end would make children align along the center vertically and along the end (right) horizontally.

Show Hide

These clasess are used to show and/or hide content by device via media queries.

Use one or combination of the available classes for toggling content across viewport breakpoints.

Classes Extra small devices (Phones)
.show-xs-* Visible
.show-sm-* Hidden
.show-md-* Hidden
.show-lg-* Hidden
.hide-xs Hidden
.hide-sm Visible
.hide-md Visible
.hide-lg Visible

.show-*--* clasess come in three variations, one for each CSS display property value:

Group of Classes CSS display
.show-*--block display: block;
.show-*--inline display: inline;
.show-*--inline-block display: inline-block;
